Phased Array Focal Law Calculator
=================================

Version: 0.1.0
Date: 2026-03-06

DESCRIPTION
-----------
This application calculates delay laws for phased array ultrasonic  testing.
It allows you to configure probes, wedges, and geometric focal points.

HOW TO RUN
----------
If you have the standalone executable:
1. Double-click "PA_Calculator.exe".

If you are running from Python source:
1. Install Python 3.x.
2. Install dependencies: pip install numpy matplotlib
3. Run: python python/app.py
