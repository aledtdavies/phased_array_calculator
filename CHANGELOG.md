# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-06
### Added
- Core Python library for Delay Law calculation (DelayLaw, Material, Probe, Wedge).
- GUI built with Tkinter for interactive phased array focal law calculation.
- Ray tracing visualization with matplotlib in GUI.
- Delay profile histogram visualization.
- CSV export capability for calculated focal laws.
- Build script (`build_executable.py`) for creating standalone executables with PyInstaller.
- Initial User Manual (`UserManual.html`).
